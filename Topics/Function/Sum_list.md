# Challenge: Sum all the numbers in list

## Description
Write a Python function to sum all the numbers in a list.

Sample List : (8, 2, 3, 0, 7)

## Solution

```python
# Provide your solution here.
def summ_list(lst):
    return sum(lst)

if __name__ == "__main__":
    result = summ_list([8, 2, 3, 0, 7])
    print(result)
